\doxysubsection{Detailed Description}
Stores all the Mouse Data. 

The Mousecharateristics Structure stores all of the bindings to the SDLK input ints. Stores all the SDLK bindings for all standard mouse buttons, as well as the mouse location. Mouse sensitivity is also set here
