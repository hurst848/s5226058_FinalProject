\doxysubsection{Detailed Description}
Is where the game engine actually runs. 

The \mbox{\hyperlink{struct_h_g_e_1_1_core}{Core}} Structure acts as the root node for the entire game engine. All crucial structure required for operation are stored here. The game loop also occurs within this structure.
