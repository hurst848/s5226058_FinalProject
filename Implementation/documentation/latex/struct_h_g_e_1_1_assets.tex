\doxysubsection{Detailed Description}
Holds all the resources used in the project. 

The assets class holds all the resources used within the game engine. These include Meshes, sounds and textures. The corresponding naming scheme can be found bellow, for naming and retriving the resources\+: ~\newline
$\ast$.snd = \mbox{\hyperlink{struct_h_g_e_1_1_sound}{Sound}} File~\newline
$\ast$.msh = \mbox{\hyperlink{struct_h_g_e_1_1_mesh}{Mesh}}~\newline
$\ast$.txtr = \mbox{\hyperlink{struct_h_g_e_1_1_texture}{Texture}}~\newline
