\doxysubsection{Detailed Description}
Is the Parent Structure to the Colliders. 

The \mbox{\hyperlink{struct_h_g_e_1_1_collider}{Collider}} structure is the parent strucure to both the \mbox{\hyperlink{struct_h_g_e_1_1_box_collider}{Box\+Collider}} and the \mbox{\hyperlink{struct_h_g_e_1_1_sphere_collider}{Sphere\+Collider}}. Additonally it stores data used by both.
