\doxysubsection{Detailed Description}
Is the parent Structure to all Components, and user made functionallity. 

The \mbox{\hyperlink{struct_h_g_e_1_1_component}{Component}} Structure is where the majority of the logic in the game engine is derived. Users can make their own components by inheriting from it. Are stored in a vector held within their respective \mbox{\hyperlink{struct_h_g_e_1_1_entity}{Entity}}.
