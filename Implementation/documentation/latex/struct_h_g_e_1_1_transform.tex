\doxysubsection{Detailed Description}
Stores all transformational data. 

The \mbox{\hyperlink{struct_h_g_e_1_1_transform}{Transform}} Structure is a component used to store all positional, rotational and scaling data. Is also where the model matrix is generated. Every created \mbox{\hyperlink{struct_h_g_e_1_1_entity}{Entity}} structure is created with a \mbox{\hyperlink{struct_h_g_e_1_1_transform}{Transform}} by default, unless manually removed
