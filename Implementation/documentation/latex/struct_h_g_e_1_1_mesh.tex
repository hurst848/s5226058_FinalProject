\doxysubsection{Detailed Description}
Stores Vertex Array objects that have mesh data. 

The \mbox{\hyperlink{struct_h_g_e_1_1_mesh}{Mesh}} Structure, derived from \mbox{\hyperlink{struct_h_g_e_1_1_resource}{Resource}}, creates and stores a Open\+GL vertex array object, to allow for rendering of the mesh file. Only .obj files are currently accepted.
