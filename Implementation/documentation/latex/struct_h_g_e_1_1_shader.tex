\doxysubsection{Detailed Description}
Stores the shader resources. 

The \mbox{\hyperlink{struct_h_g_e_1_1_shader}{Shader}} Structure, derived from \mbox{\hyperlink{struct_h_g_e_1_1_resource}{Resource}}, takes a valid .vert or .frag file and parses it for use in the \mbox{\hyperlink{struct_h_g_e_1_1_shader_program}{Shader\+Program}}. Uniform and Attribute flags are also parsed here, details of what each flag is and is correponing varible can be found bellow\+: ~\newline
fa-\/1 = attribute vec3 a\+\_\+\+Position ~\newline
fa-\/2 = attribute vec2 a\+\_\+\+Texture\+Coordinates ~\newline
fa-\/3 = attribute vec3 a\+\_\+\+Normal ~\newline
fu-\/1 = uniform mat4 u\+\_\+\+Model\+Matrix ~\newline
fu-\/2 = uniform mat4 u\+\_\+\+Projection\+Matrix ~\newline
fu-\/3 = uniform mat4 u\+\_\+\+View\+Matrix ~\newline
fu-\/4 = uniform vec3 u\+\_\+\+Light\+Colour ~\newline
fu-\/5 = uniform vec3 u\+\_\+\+Light\+Position ~\newline
fu-\/6 = uniform float u\+\_\+\+Light\+Ambient\+Strength ~\newline
fu-\/7 = unifrom sampler2D u\+\_\+\+Texture ~\newline
~\newline
~\newline
 Addtionally, the format of the top of each shader file should start with \char`\"{}//$\sim$\char`\"{} and must be space apart with commas. Example shown bellow\+: ~\newline
\char`\"{}//$\sim$ fu-\/4, fu-\/5, fu-\/6\char`\"{}
