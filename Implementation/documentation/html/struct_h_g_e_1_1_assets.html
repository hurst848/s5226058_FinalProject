Detailed Description

Holds all the resources used in the project.

The assets class holds all the resources used within the game engine. These include Meshes, sounds and textures. The corresponding naming scheme can be found bellow, for naming and retriving the resources:
*.snd = Sound File
*.msh = Mesh
*.txtr = Texture
