Detailed Description

Stores all the Key Data.

The Keys Structure stores all of the bindings to the SDLK input ints. Used in the Inputs Class. FUNC refers to a keyboard function, SYM refers to a keyboard symbol, NUM refers to a number, KEYPAD referes to a key on the number pad
